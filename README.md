# ecommerce-sales-analysis
# 📊 Analisis Tren Penjualan E-Commerce

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis data penjualan e-commerce guna menemukan pola, tren, dan insight yang dapat membantu pengambilan keputusan bisnis. Analisis dilakukan berdasarkan kategori produk, wilayah, metode pembayaran, serta total penjualan.

## 🎯 Tujuan Analisis
- Mengidentifikasi kategori produk dengan penjualan tertinggi
- Menganalisis tren penjualan berdasarkan waktu
- Mengetahui metode pembayaran yang paling sering digunakan
- Menganalisis distribusi penjualan berdasarkan wilayah
- Memberikan insight untuk meningkatkan strategi bisnis

## 🗂️ Dataset
Dataset yang digunakan memiliki beberapa kolom utama, antara lain:
- Transaction ID
- Date
- Product Category
- Product Name
- Units Sold
- Unit Price
- Total Revenue
- Region
- Payment Method

## 🛠️ Tools & Teknologi
- Microsoft Excel (Data Cleaning & Analisis)
- SQL (Query data dan eksplorasi data)
- (Opsional) Python (Pandas, Matplotlib)
- GitHub (Repository & Dokumentasi)

## 📊 Proses Analisis
1. Data Cleaning  
   Menghapus data duplikat, menangani missing values, dan memastikan format data konsisten.

2. Exploratory Data Analysis (EDA)  
   Menganalisis distribusi data, tren penjualan, dan hubungan antar variabel.

3. Data Visualization  
   Membuat grafik untuk mempermudah pemahaman data seperti:
   - Bar chart penjualan per kategori
   - Line chart tren penjualan
   - Pie chart metode pembayaran

4. Insight & Kesimpulan  
   Menarik kesimpulan dari hasil analisis untuk memberikan rekomendasi bisnis.

## 📈 Hasil & Insight
- Kategori produk tertentu memiliki kontribusi terbesar terhadap total revenue
- Metode pembayaran digital lebih dominan dibandingkan metode lainnya
- Wilayah tertentu menunjukkan performa penjualan yang lebih tinggi
- Terdapat tren peningkatan/penurunan penjualan pada periode tertentu

## 💡 Rekomendasi
- Fokus pada kategori produk dengan performa tinggi
- Mengoptimalkan strategi pemasaran di wilayah dengan penjualan rendah
- Meningkatkan penggunaan metode pembayaran yang populer
- Melakukan analisis lanjutan untuk prediksi penjualan
